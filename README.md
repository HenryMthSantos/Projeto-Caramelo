# 📊 Projeto Caramelo - Análise de Vendas com Python

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de realizar uma análise exploratória de dados de vendas utilizando Python.

A análise busca identificar padrões de consumo, desempenho das lojas, categorias mais vendidas, ticket médio e comportamento temporal das vendas.

O projeto foi construído utilizando bibliotecas voltadas para análise e visualização de dados, permitindo transformar informações brutas em insights relevantes.

---

* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
